# GreyPanel v2

Современная панель управления игровыми серверами с форумом, VIP-системой, мониторингом, новостной лентой и тикет-системой.

![PHP](https://img.shields.io/badge/PHP-8.4+-blue)
![Symfony](https://img.shields.io/badge/Symfony-7.2-black)
![License](https://img.shields.io/badge/license-MIT-green)

## Возможности

- **Модульная архитектура**: лёгкое расширение функционала.
- **Аутентификация и роли**: пользователь, модератор, администратор, root с гибкими правами.
- **Управление серверами**:
  - Интеграция привилегий через `users.ini`, AmxBans/CsBans, SourceBans, FTP.
  - Статистика игроков (CSStats, AES, RankMe, Levels Ranks).
  - Автоматический импорт бан-листа из внешней БД AmxBans/SourceBans.
  - Мониторинг серверов через Source Query.
- **Форум**:
  - Полноценный форум с категориями и разделами.
  - Markdown-редактор (EasyMDE) с предпросмотром, загрузкой изображений, эмодзи.
  - Лайки, цитирование, отслеживание прочитанного.
- **Новости**: Markdown-посты с комментариями и загрузкой изображений.
- **Тикет-система**: поддержка категорий (включая заявки на разбан).
- **Покупка VIP-привилегий**: активация через FTP, MySQL или AmxBans.
- **Баланс пользователей**, реферальная программа.
- **Приём платежей** через ЮMoney.
- **Онлайн-чат** на главной странице.
- **Twig + Bootstrap 5 + Alpine.js** – современный фронтенд.
- **Сборка фронтенда** через Vite.
- **Консольные команды** (Symfony Console).
- **Веб-установщик** с проверкой окружения и самоудалением.
- **Мультиязычность**(русский, английский) с возможностью добавления новых языков.
- **REST API** для виджетов (последние темы, топ донатеров, онлайн).
- **Полноценная панель администратора** со своей темой и редактором шаблонов.
- **Безопасность**: CSRF-защита, rate-limiting, шифрование паролей и конфиденциальных данных.

## Стек технологий

- **PHP** 8.4+
- **Symfony Components** (HttpFoundation, Cache, Console, Translation, Serializer, Validator, Lock, RateLimiter, HtmlSanitizer)
- **FastRoute** (роутинг) – в процессе миграции на Symfony Routing
- **Twig** (шаблонизатор)
- **Bootstrap 5**, **Alpine.js**, **Chart.js**, **SortableJS**, **EasyMDE**, **CodeMirror**
- **MySQL** / **MariaDB**
- **Vite** (сборка JS / SCSS)

## Требования

- **PHP** >= 8.4 (расширения: `pdo_mysql`, `mbstring`, `ftp`, `fileinfo`, `openssl`, `gd|Imagick`)
- **MySQL** >= 5.7
- **Composer**
- **Node.js** + **npm** (для сборки фронтенда)
