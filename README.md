# GreyPanel v2

> **Статус:** активная разработка. Не готов к продакшену.  
> Современная панель управления серверами с форумом, VIP-системой, мониторингом и встроенным установщиком.

## О проекте

GreyPanel — веб-приложение для администрирования игровых серверов.  
Версия 2.0 написана с использованием компонентов Symfony, собственного DI-контейнера и шаблонизатора Twig.

**Ключевые возможности:**
- Модульная архитектура
- Аутентификация, роли, middleware
- Управление VIP-серверами через Source Query / RCON и FTP
- Мониторинг серверов в реальном времени
- Встроенный форум с BB-кодами, лайками и системой прочтения
- Приём платежей (Внедряются)
- Система банов и платного разбана
- Баланс пользователей, реферальная программа
- Twig + Bootstrap 5 + Alpine.js
- Сборка фронтенда через Vite
- Консольные команды (Symfony Console)
- **Веб-установщик** с проверкой окружения и самоудалением

## Требования

- **PHP** >= 8.5 (расширения: `pdo_mysql`, `mbstring`, `ftp`, `fileinfo`, `openssl`, `gd|ImageMagick`)
- **MySQL** >= 5.7
- **Composer**
- **Node.js** + **npm**
