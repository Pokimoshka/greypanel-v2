# GreyPanel v2

> **Статус:** активная разработка, готов к тестированию.  
> Современная панель управления игровыми серверами с форумом, VIP-системой, мониторингом, новостной лентой и тикет-системой.

## О проекте

GreyPanel — веб-приложение для администрирования игровых серверов.  
Версия 2.0 написана с использованием компонентов Symfony, собственного DI-контейнера и шаблонизатора Twig.

**Ключевые возможности:**
- Модульная архитектура
- Аутентификация, роли (пользователь, модератор, админ, root), middleware
- Управление серверами: настройка интеграций привилегий (users.ini, AmxBans/CsBans) и статистики (CSStats, AES)
- Автоматическое подключение бан-листа из внешней БД AmxBans/CsBans
- Публичный мониторинг серверов через Source Query
- Встроенный форум с Markdown-редактором (EasyMDE), лайками и системой прочтения
- Новостная лента с поддержкой Markdown и загрузкой изображений
- Тикет-система (поддержка) с категориями (включая заявки на разбан)
- Покупка VIP-привилегий (активация через FTP, MySQL или AmxBans)
- Баланс пользователей, реферальная программа
- Приём платежей (ЮMoney)
- Чат на главной странице
- Twig + Bootstrap 5 + Alpine.js
- Сборка фронтенда через Vite
- Консольные команды (Symfony Console)
- Веб-установщик с проверкой окружения и самоудалением

## Требования

- **PHP** >= 8.5 (расширения: `pdo_mysql`, `mbstring`, `ftp`, `fileinfo`, `openssl`, `gd|ImageMagick`)
- **MySQL** >= 5.7
- **Composer**
- **Node.js** + **npm**
